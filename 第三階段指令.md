# 第三階段指令｜教學對話 + 對戰系統 + 破關流程

## 前提
第一、二階段已完成：領地、建築互動、傳送門、時代場景、關卡選單。

## 給 Claude Code 的指令（直接複製貼上）

**本次完成遊戲的核心玩法：**

---

### 項目一：知識教學對話系統

點擊關卡選單中的關卡後觸發：

- 畫面切換到對應時代場景
- 一名隨從武士從右側走出，停在畫面中央
- 顯示聊天對話框（像漫畫對話泡泡的像素風格）
- 對話框內容：根據 questions.json 中該關的 `stageName` 顯示對應的教學說明
  - 例如：stageName 是「Claude是什麼？」→ 顯示「幸村大人，本關將學習 Claude AI 的基本概念...」
  - 教學內容可簡短摘要該關的學習重點（2-3句話）
- 玩家點擊「了解！開始對戰」按鈕 → 進入對戰畫面

---

### 項目二：對戰畫面系統

**橫屏佈局：**
```
[左上：答題數/總題數]          [右上：撤退按鈕]
[敵人名字]                     [真田幸村]
[敵人血條]                     [血條 100HP]
[敵人像素角色]                 [真田幸村像素角色]
         [題目與選項區域（中央下半部）]
                               [右下：道具欄]
```

**直屏佈局（手機豎拿）：**
```
[左上：答題數/總題數]  [右上：撤退]
[敵人名字][敵人血條]
[敵人角色]   [幸村角色]
[我方名字][我方血條]
[題目區域]
[選項區域]
[道具欄]
```

**血量機制：**
- 每關開始時血量自動補滿至 100 HP
- 每題扣血量 = 100 ÷ 該關總題數（無條件捨去）
- 答對 → 敵人扣等量血量，播放攻擊動畫
- 答錯 → 我方扣等量血量，播放受傷動畫，顯示正確答案和解釋

**題目顯示：**
- 從 questions.json 讀取該關卡的題目
- 選擇題：顯示4個選項按鈕，點擊作答
- 填空題：題目有空白處，顯示4個詞語按鈕，點擊填入

**答題後：**
- 答對：顯示綠色「✓ 正確！」提示 1 秒，進入下一題
- 答錯：顯示紅色「✗ 答錯了！」+ 正確答案 + explanation 內容，2 秒後進入下一題

**道具欄（右下角）：**
- 顯示背包中的道具數量
- 點擊道具圖示 → 立即使用，回復對應血量
- 使用後道具數量 -1

**撤退：**
- 點擊撤退按鈕 → 確認對話框「確定要撤退？關卡進度不保留。」
- 確認 → 返回領地

---

### 項目三：勝負判定

**失敗條件：**
- 我方血量歸零 → 顯示「撤退！」動畫 → 返回領地
- 道具可在失敗前使用補血救回

**勝利條件：**
- 所有題目作答完畢且血量 > 0 → 進入破關畫面

---

### 項目四：破關畫面

顯示以下內容：
- 「關卡完成！」大標題
- 星評（依剩餘血量）：
  - 81-100 HP → ⭐⭐⭐
  - 41-80 HP → ⭐⭐
  - 1-40 HP → ⭐
- 掉落寶物（依機率計算）：
  - 銅幣（15-35枚，必掉）
  - 蕎麥麵（20% 機率）
  - 烤魚（10% 機率）
  - 天婦羅蓋飯（5% 機率）
  - 以上道具只掉一種（65% 機率無道具）
  - **大魔王關**：另外必掉時代寶石 + 80-120 銅幣 + 50% 機率天婦羅蓋飯
- 將獲得的銅幣和道具加入存檔
- 解鎖下一關
- 更新該關最高星評（只有更高才覆蓋）
- 選單按鈕：
  - 「繼續下一關」（若有下一關）
  - 「回到領地」
