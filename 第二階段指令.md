# 第二階段指令｜建築互動 + 傳送門 + 時代場景

## 前提
第一階段已完成：領地場景、角色移動、建築外觀、存檔系統。

## 給 Claude Code 的指令（直接複製貼上）

**本次只完成以下四個項目：**

---

### 項目一：建築互動功能

#### 真田屋（可進入）
- 靠近並按 E / 觸控互動 → 觸發進入動畫（畫面淡出淡入）
- 進入後顯示屋內場景：榻榻米、床墊、盔甲座、武士刀座
- 屋內有「離開」按鈕可返回領地

#### 村正鐵匠鋪（不可進入）
- 靠近並按 E / 觸控互動 → 顯示對話框
- 鐵匠說：「幸村，你的使命配得上村正家的刀。」
- 點擊對話框關閉

#### 雜貨舖（可進入）
- 靠近並按 E / 觸控互動 → 進入商店場景
- 老闆站在櫃檯後，靠近老闆 → 開啟購買介面
- 商品清單：
  - 蕎麥麵：+15 HP，售價 30 銅幣
  - 烤魚：+30 HP，售價 60 銅幣
  - 天婦羅蓋飯：+50 HP，售價 100 銅幣
- 購買成功 → 扣除銅幣，道具加入背包
- 銅幣不足 → 顯示「銅幣不足」提示
- 畫面右上角顯示當前銅幣數量

---

### 項目二：石製傳送門互動

- 靠近並互動 → 開啟選單：
  1. 進入三國時代（Claude AI）
  2. 進入騎士時期（Claude Cowork）
  3. 進入江戶時期（Claude Code）
  4. 放入寶石（只有已獲得寶石時才顯示此選項）
- 選擇時代 → 畫面變黑 → 場景切換到對應時代
- 放入寶石時：門框漩渦從紫色變金色持續 1 秒，再恢復紫色
- 三顆都放入 → 預留結局觸發點（本階段先做空白頁面即可）

---

### 項目三：三個時代場景

每個時代的基本場景（本階段只需要外觀，關卡功能下一階段做）：

#### 三國時期
- 左側：石製傳送門（互動只有「回到領地」）
- 右側：三國風格城堡，有軍旗和隨機走動的兵將
- 背景顏色基調：土黃色、紅色

#### 騎士時期
- 左側：石製傳送門
- 右側：中世紀貴族城堡，有弓箭手和騎士走動
- 背景顏色基調：灰色、藍色

#### 江戶時期
- 左側：石製傳送門
- 右側：以岡崎城為設計的建築，有武士和馬
- 背景顏色基調：深綠色、棕色

---

### 項目四：關卡選單

靠近各時代右側城堡 → 開啟關卡選單：
- 顯示該時代的 5 個關卡
- 已解鎖的關卡：顯示關卡名稱、敵將名稱、難度（1-5顆星）、最高星評（0-3顆星）
- 未解鎖的關卡：顯示鎖頭圖示
- 第一關預設解鎖，其餘需破前一關才解鎖
- 點擊已解鎖關卡 → 預留進入關卡的觸發點（本階段先顯示「功能建設中」）
