### 項目一：開場動畫

遊戲第一次啟動時（或從標題畫面開始時）播放：

共 6 個畫面，每個畫面 5 秒，畫面之間淡入淡出：

1. 黑色背景 + 像素風戰場圖 → 文字：「慶長二十年，大阪夏之戰。」
2. 真田幸村倒下的像素圖 → 文字：「真田幸村，最後一戰，力竭而亡。」
3. 光芒從黑暗中透出的效果 → 文字：「然而，翌日清晨，他睜開了雙眼。」
4. 領地全景像素圖，傳送門在遠方發光 → 文字：「一個陌生的領地，一扇神秘的石門，等待著他。」
5. 傳送門特寫，三個寶石缺口清晰可見 → 文字：「門上有三個缺口，據說集齊三顆寶石，便能回到故鄉。」
6. 真田幸村握刀站立 → 文字：「真田幸村的新征途，現在開始。」→ 淡入領地主畫面

- 右下角加「跳過」按鈕，點擊可直接進入領地
- 動畫播完或跳過後，之後重啟遊戲不再顯示（存入 localStorage）

- 打敗三國大魔王（曹操）後：在扎營處插入三國時代旗幟（紅底黑字「魏」）
- 打敗騎士大魔王（亞瑟王）後：插入騎士時代旗幟（藍底金色十字）
- 打敗江戶大魔王（德川家康）後：插入江戶時代旗幟（黑底金色葵紋）

### 項目三：背景音樂系統

各場景播放對應風格的背景音樂（使用 Web Audio API 生成像素風格音效，不需要外部音樂檔案）：

- 領地主畫面：輕鬆的日式和風像素音樂
- 三國時期場景：古箏、琵琶風格的像素音樂
- 騎士時期場景：管風琴、魯特琴風格的像素音樂
- 江戶時期場景：尺八、太鼓風格的像素音樂
- 對戰畫面：緊張的戰鬥節奏像素音樂
- 破關畫面：勝利的短促像素音效

- 右上角加「音樂開/關」按鈕，設定存入 localStorage

### 項目四：寶石收集與結局

**傳送門放入寶石：**
- 已獲得寶石時，傳送門選單顯示「放入寶石」選項
- 選擇後顯示已有的寶石，點擊放入
- 放入動畫：漩渦從紫色變金色 1 秒，恢復紫色
- 存入 localStorage

**集齊三顆寶石的結局：**

- 點擊傳送門 → 觸發結局動畫：

  2. 文字：「三顆寶石齊聚，石門緩緩開啟。」

  4. 文字：「幸村，你完成了不可能的征途。」

  7. 顯示「THE END」+ 「重新遊玩」按鈕
