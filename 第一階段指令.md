# 第一階段指令｜基礎架構 + 領地主畫面

## 給 Claude Code 的指令（直接複製貼上）

請閱讀專案資料夾中的 CLAUDE.md 和 questions.json，了解整個遊戲的規格。

**本次只完成以下三個項目，其他功能先不做：**

---

### 項目一：基礎架構

建立遊戲的基本骨架：
- 建立 `index.html`、`style.css`、`game.js` 三個主要檔案
- CSS 設定像素風格基礎（使用 `image-rendering: pixelated`）
- 建立場景管理系統：可以切換不同畫面（開場動畫、領地、對戰等）
- 建立響應式框架：能偵測橫屏／直屏並套用不同CSS

---

### 項目二：資料讀取系統

- 讀取 `questions.json`
- 建立資料管理模組，讓其他功能可以查詢題目、關卡資訊
- 建立 localStorage 存檔系統，儲存以下內容：
  - 各關卡解鎖狀態與最高星評
  - 當前銅幣數量
  - 道具背包（蕎麥麵、烤魚、天婦羅蓋飯各自的數量）
  - 已獲得的寶石清單（翡翠、紅寶石、黃玉）
  - 軍隊扎營處旗幟狀態（三個時代各一）

---

### 項目三：領地主畫面

建立真田幸村的領地場景：

**畫面需求：**
- 2D 橫向像素風場景
- 玩家可以左右移動真田幸村（鍵盤方向鍵 或 手機觸控按鈕）
- 真田幸村外觀：紅色盔甲、頭盔上有鹿角的像素角色

**五個建築（先做外觀和靠近觸發，互動功能下一階段完成）：**
1. 真田屋：江戶武家敷屋風格，屋簷有六文錢家徽
2. 村正鐵匠鋪：開放式小舖，有鐵匠在門口，招牌寫「村正」
3. 雜貨舖：江戶時期風格小舖
4. 軍隊扎營處：有帳篷和火堆，有幾名武士走動
5. 石製傳送門：石頭門框，門框內有半透明紫色漩渦效果

**靠近互動：**
- 當角色走到建築附近時，建築上方顯示像素風文字的建築名稱
- 顯示「按 E 互動」或觸控按鈕的提示
