### 教學場景火焰效果

#### 火焰粒子系統
- **粒子類型**：灰色上升粒子，隨機位置生成
- **生命週期**：60幀（約1秒）
- **顏色混合**：
  - 橘色：`#FF6B35` (60%)
  - 黃色：`#FFA500` (25%)
  - 紅色：`#FFD700` (15%)
- **粒子速度**：
  - 水平速度：`-1 ~ 1`
  - 垂直速度：`-1.5 ~ -0.5`（向上飄）

#### 像素火焰層
```javascript
const colors=['#FF6B35', '#FFA500', '#FFD700'];
// 3層火焰，每層高度約25像素，寬度隨高度變化
```

#### 發光效果
- 基礎發光大小：55像素
- 呼吸動畫幅度：±12像素
- 發光顏色：`#FF8C00`（深橘色）

### 教學場景紮營處改進

#### 帳篷設計（真田領地風格）
```javascript
// 帳篷框架（深色）
ctx.fillStyle=col.buildDark;

// 帳篷頂部（中色斜頂）
ctx.fillStyle=col.buildMid;

// 邊框強調
ctx.strokeStyle=col.buildDark;
ctx.lineWidth=1;
```

#### 燈籠效果
- 位置：帳篷左側（X偏移 -10）
- 發光大小：8 ~ 28像素（動態呼吸）
- 顏色：隨時代變化（`col.accent2`）
- 燈籠本體：6×8像素矩形

#### 紮營處色彩系統
- **三國時期**：使用 `eraColors.ai`
- **騎士時期**：使用 `eraColors.cowork`
- **江戶時期**：使用 `eraColors.code`
